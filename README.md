📄 PDF Sorter / Indexer  
Secure multi-user document indexing system for managing and searching PDF collections.

Features:
- JWT authentication with refresh token rotation
- Multi-user document isolation
- PDF upload and text extraction
- Full-text search with contextual snippets
- Tag-based organization
- Bulk export of documents as ZIP

Built with a modular backend architecture (NestJS + Prisma + SQLite) designed to support asynchronous processing and scalable document indexing.
